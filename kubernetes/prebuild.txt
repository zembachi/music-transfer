1) включить аддон ingress для minikube
minikube addons enable ingress

2) установить сертификат для keycloak
kubectl create secret tls music-transfer-tls-secret --cert keycloak-cert.pem --key keycloak-key.pem

3) в миникубе создать неймспейс для keycloak
kubectl create namespace security

4) создать секрет для БД кейклока (пароли можно поменять для секьюрности)
kubectl -n security create secret generic keycloak-postgresql --from-literal=password=keycloak --from-literal=postgres-password=keycloak

5) создать секрет для кейклока (логин и пароль можно поменять)
kubectl -n security create secret generic keycloak-admin-creds --from-literal=username=admin --from-literal=password=password
